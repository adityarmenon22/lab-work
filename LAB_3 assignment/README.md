# Portfolio Website

This is a simple portfolio website for my Web Development Lab assignment.

## Files

- `index.html` — Main website file
- `styles.css` — All the styling

## How to Run

1. Make a new folder and add both files.
2. Open `index.html` in any web browser (like Chrome or Edge).
3. You will see your portfolio page with styles applied.

## What it does

- Shows basic info about me.
- Lists my skills and a few sample projects.
- Has a simple contact section (not working, just for display).

## Features

- Responsive design that works on mobile and desktop
- Simple navigation menu
- About section with statistics
- Skills section with different categories
- Projects showcase with images
- Contact form
- Footer with links

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
